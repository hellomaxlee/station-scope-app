# StationScope
### **App link: https://station-scope.up.railway.app/**

- StationScope is a desktop/mobile app that turns a work address into a personalized, data-driven NYC housing guide.
- Get mapped stations, census insights, and a tailored AI report in minutes!
- Using the app requires no familiarity of NYC geography - the app has a built in interactive map thanks to OpenStreetMap.
- The concept originated at a hackthon (https://github.com/hellomaxlee/hack-nyu).
- Let me know of any suggestions, bugs, or if you want to collaborate!
